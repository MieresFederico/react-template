# react-template

An awesome React template to jumpstart your projects!

## Description

React project with the best guidelines for code maintenance

## Getting Started

### Dependencies

* node v18.13.0

### Installing
```
* npm i
```

### Executing program

#### Dev
```
* npm run dev
```

#### Prod
```
* npm run build
```

## Technologies

* Prettier
* Lint-staged

* Vitest
